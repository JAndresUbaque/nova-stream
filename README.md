# NovaStream 🎥

Plataforma de streaming ficticia construida **únicamente con HTML5, CSS3 y JavaScript Vanilla (ES6+)**.

Proyecto creado como portafolio profesional para demostrar sólidos fundamentos de Front-End.

## ✨ Características

- Diseño futurista con Glassmorphism
- Dark Theme premium
- Responsive (Mobile First)
- Carruseles interactivos
- Sistema de favoritos con LocalStorage
- Consumo de API TMDB

## 🛠 Tecnologías

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Glassmorphism)
- JavaScript ES6+ (Fetch API, Async/Await, DOM Manipulation)
- Git & GitHub
